Capstone Project For the Data Science Specialisation

Prediction of Next Word

Wanda Ken 01/07/2021

How the App has been built

  • The Application is about predicting the next word from an input text.

  • It has been built using R Shiny package in RStudio and published on the Shinyapps.io server.

  • The Application consists of two files:

    • ui.R which controls the interface and inputs of the application.
    • server.R which controls what the application does.
    • The source codes for the files have been loaded in Github link https://github.com/Wanda173/Capstone

  • The application can be run using the link: (It can take a few seconds to load) https://wandaken.shinyapps.io/predict/

The Dataset

  • The dataset used was downloaded from SwiftKey.

  • The data is made up of 3 corpus - en_US.blogs, en_US.news and en_US.twitter, each file being more than 150 MB.

  • Random Sampling of 5% of each file was used as training set.

  • The sample was cleaned and tokenized into quadgrams, trigrams, bigrams and unigrams.

  • The processing of the sample in chunks of lines improved the performance of the tokenization process.

How the Prediction Algorithm works

  • The Stupid Backoff algorithm has been used to predict the next word from the generated tokens.

  • Depending on the level of n-gram, a score is calculated for each word retrieved.

  • The search of the next word first starts to quadgrams, then goes to trigrams and finally bigrams.

  • If no word is found, the search continues to the unigrams.

  • The word that comes with the highest score after ordering the top ten words obtained will be the next predicted word.

Application Input/Output Instructions

  • On the Side bar panel, at least one word or phrase must be entered. The input does not process any symbol, numbers or punctuations entered alone as a single word.

  • On Clicking the Submit button, the next predicted word will be displayed at the bottom of the first tab of the main panel.

  • On the first tab of the main panel "Plot", a bar chart will appear.

    • The plot shows the top 10 highest score words each represented by a bar.
    • The color of the bar distinguishes from which n-gram the next word was retrieved as shown by the legend.
    • A text output will show what is the next word at the bottom.

  • On the second tab "Data", a tabular representation of the data is shown to give an idea of how the word fits in the n-gram token.

  • The third tab "About" gives some basic information about the App.
